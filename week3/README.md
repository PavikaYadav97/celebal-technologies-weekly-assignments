# customer-intelligence-system-ml
# Customer Intelligence System using Classification, Ensemble Learning & Clustering

## Project Overview

This project develops a Customer Intelligence System using clustering, classification, and ensemble learning techniques.

The workflow includes:

- Data preprocessing
- Exploratory Data Analysis (EDA)
- Feature scaling
- Customer segmentation using clustering
- Classification modeling
- Ensemble learning
- Model evaluation
- Feature importance analysis
- Business insights generation

---

## Dataset

Country Development Dataset

Features include:

- Child Mortality
- Exports
- Health
- Imports
- Income
- Inflation
- Life Expectancy
- Total Fertility
- GDP per Capita

---

## Techniques Used

### Clustering
- K-Means Clustering
- DBSCAN

### Classification
- Logistic Regression

### Ensemble Learning
- Random Forest
- XGBoost

---

## Evaluation Metrics

- Accuracy
- Precision
- Recall
- F1 Score
- Silhouette Score

| Random Forest | 0.971 | 0.975 | 0.971 | 0.971 |

## Key Insights

- Child mortality emerged as the strongest segmentation factor.
- GDP per capita and income significantly contributed to classification.
- Ensemble methods achieved the highest performance.
- Clustering revealed distinct socio-economic country groups.

---

## Conclusion

This project demonstrates an end-to-end machine learning workflow integrating clustering, classification, ensemble learning, and insight generation for customer intelligence analysis.
